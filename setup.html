4 Install Dependencies
Pending

Install Python packages required for the backend.

cd backend pip install -r requirements.txt

This installs: Flask, Flask-CORS, Supabase, NumPy, Pandas, python-dotenv

5 Get API Keys
Pending

OpenRouter API Key (for AI features):

  • Sign up at OpenRouter.ai (FREE)
  • Go to Keys section and create a new key
  • Copy the key (starts with sk-or-v1-...)

Supabase Credentials (for backend):

  • Create account at Supabase.com
  • Create a new project
  • Go to Settings ‚Üí API
  • Copy: Project URL, anon key, and service_role key
Get OpenRouter Key Get Supabase Keys
6 Configure Frontend
Pending

Create configuration file with your API keys.

cd frontend Copy-Item config.example.js config.js

Edit frontend/config.js and add your keys:

window.CONFIG = { OPENROUTER_API_KEY: 'sk-or-v1-YOUR-KEY-HERE', SUPABASE_URL: 'https://xxxxx.supabase.co', SUPABASE_ANON_KEY: 'your-anon-key', BACKEND_URL: 'http://localhost:5000' };
7 Configure Backend
Pending

Create backend environment file.

cd backend Copy-Item .env.example .env

Edit backend/.env and add:
